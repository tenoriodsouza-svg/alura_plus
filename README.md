# 🚀 Alura Plus - Landing Page

Projeto de uma landing page desenvolvido durante treinamento da Alura, com foco na construção de interfaces modernas utilizando HTML e CSS.

---

## 📌 Sobre o projeto

O **Alura Plus** é uma página fictícia desenvolvida como parte de um projeto prático, com o objetivo de aplicar conceitos essenciais de desenvolvimento front-end na construção de uma interface real.

Durante o desenvolvimento, foram aplicados conhecimentos de estruturação com HTML e estilização com CSS, além da interpretação de layouts a partir de protótipos no Figma.

---

## 💻 Tecnologias utilizadas

- HTML5  
- CSS3  

---

## 🎯 Objetivo

Este projeto teve como principais objetivos:

- Construir uma landing page utilizando HTML e CSS  
- Aplicar conceitos de layout com base em protótipos do Figma  
- Praticar a organização e estruturação de código  
- Criar e versionar o projeto com Git e GitHub  
- Realizar o deploy da aplicação utilizando GitHub Pages
